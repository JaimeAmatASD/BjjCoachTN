# **ChatGPT Personalizado para BJJ y Grappling**

## **Descripción General**
Este proyecto implementa un ChatGPT especializado en la creación de **planes de nutrición y entrenamiento personalizados** enfocados en preparar a los atletas de **Brazilian Jiu-Jitsu (BJJ)**, **grappling** y disciplinas relacionadas para competencias. 

### **Características Principales**
1. **Preparación para Torneos:**
   - Planificación de entrenamientos progresivos adaptados a fechas clave.
   - Planes de nutrición optimizados para corte de peso, mantenimiento o ganancia muscular.
2. **Planes Personalizados:**
   - Rutinas semanales, mensuales, circuitos y periodización avanzada.
   - Recomendaciones nutricionales diarias y semanales con tablas específicas para suplementos.
3. **Seguimiento de Progreso:**
   - Registro de fuerza, resistencia y composición corporal.
   - Tablas detalladas para monitorear tu avance hacia los objetivos.
4. **Exportación de Datos:**
   - Posibilidad de descargar planes y progresos en formato Excel para análisis más detallado.
5. **Recursos Educativos:**
   - Acceso a fuentes confiables sobre entrenamiento, nutrición y estrategias para BJJ y grappling.
