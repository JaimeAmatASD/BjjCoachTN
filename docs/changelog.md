# **Registro de Cambios (Changelog)**

Este archivo documenta todas las actualizaciones y mejoras realizadas en el proyecto ChatGPT Personalizado. Sigue el formato de versiones para rastrear los cambios de manera organizada.

---

## **[Versión 1.0.0] - Lanzamiento Inicial**
**Fecha:** DD-MM-AAAA

### **Nuevas Funciones**
- Implementación del modelo ChatGPT personalizado para:
  - Generar planes de entrenamiento y nutrición personalizados.
  - Exportar planes en formato Excel.
  - Ofrecer acceso a fuentes confiables mediante Knowledge.
- Creación de los archivos `.md`:
  - `plan_nutricion.md`: Modelos de nutrición diarios, semanales, y tablas de suplementos.
  - `rutinas_instrucciones.md`: Rutinas semanales, mensuales, circuitos y periodización.
  - `fuentes_links.md`: Lista de fuentes confiables para nutrición, entrenamiento y grappling.
  - `guia_uso_chatgpt.md`: Guía para interactuar con ChatGPT.
- Integración del enlace destacado a [www.bulk.com](https://www.bulk.com).

---

## **[Versión 1.1.0] - Mejora de Tablas y Funcionalidades**
**Fecha:** DD-MM-AAAA

### **Mejoras**
- **Nutrición:**
  - Añadida columna de micronutrientes a las tablas de nutrición (Vitamina C, Calcio, Hierro).
  - Nueva tabla detallada para suplementos, con dosis, horarios y beneficios.
- **Entrenamiento:**
  - Directrices para personalizar rutinas según nivel (principiante, intermedio, avanzado).
  - Ejemplos adicionales en cada tipo de rutina (semanal, circuito, periodización).
  - Nueva sección de movilidad y recuperación.
- **General:**
  - Instrucciones más claras en `guia_uso_chatgpt.md`.

## **[Versión 1.2.0] - Ajustes de Exportación y Feedback**
**Fecha:** DD-MM-AAAA

### **Cambios**
- Añadida opción de exportar tablas personalizadas desde Knowledge.
- Optimización de las instrucciones para ajustar planes según el feedback del usuario.
- Nuevas recomendaciones de progresión en la sección de rutinas:
  - Incremento gradual de cargas basado en %RM.
  - Ejercicios clave para test de progreso (fuerza máxima, resistencia, movilidad).
